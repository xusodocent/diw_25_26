    El grid

    En Tailwind CSS, quan utilitzem el sistema Grid, podem definir fàcilment el nombre de files i columnes que volem tenir en el nostre disseny. El sistema de grid de Tailwind CSS es basa en la utilització de classes utilitàries per crear dissenys de diverses columnes i files de manera flexible.

    1. Crear un grid bàsic

    El sistema Grid de Tailwind et permet dividir l'espai en una matriu de files i columnes. Utilitzant classes com grid-cols-{n} i grid-rows-{n}, pots especificar el nombre de columnes i files que vols.

    Exemple bàsic:

    HTML
    1
2
3
4
5
    <div class="grid grid-cols-3 gap-4">
  <div class="bg-blue-500 p-4">Columna 1</div>
  <div class="bg-green-500 p-4">Columna 2</div>
  <div class="bg-red-500 p-4">Columna 3</div>
</div>

    Explicació:

    • grid: Estableix que aquest contenidor utilitzarà el sistema de grid.
    • grid-cols-3: Defineix que hi haurà 3 columnes en aquest contenidor. Així, els 3 elements fills es distribuiran en 3 columnes iguals.
    • gap-4: Afegix un espai entre les columnes i files, en aquest cas, de 1 rem (16px).

    2. Definir el nombre de columnes (grid-cols-{n})

    Tailwind CSS et permet definir el nombre de columnes en un grid amb la classe grid-cols-{n}. On {n} és el nombre de columnes que vols tenir.

    Valors per a les columnes:

    • grid-cols-1: 1 columna
    • grid-cols-2: 2 columnes
    • grid-cols-3: 3 columnes
    • grid-cols-4: 4 columnes
    • grid-cols-5: 5 columnes
    • grid-cols-6: 6 columnes
    • grid-cols-12: 12 columnes (molt comú en dissenys de tipus "container fluid")

    Tailwind també permet classes amb valors automàtics com grid-cols-auto, que fa que el nombre de columnes sigui automàtic, adaptant-se al contingut.

    Exemple amb 4 columnes:

    HTML
    1
2
3
4
5
6
    <div class="grid grid-cols-4 gap-4">
  <div class="bg-blue-500 p-4">Columna 1</div>
  <div class="bg-green-500 p-4">Columna 2</div>
  <div class="bg-red-500 p-4">Columna 3</div>
  <div class="bg-yellow-500 p-4">Columna 4</div>
</div>

    3. Definir el nombre de files (grid-rows-{n})

    De manera similar a les columnes, també pots definir el nombre de files amb la classe grid-rows-{n}. Aquesta classe defineix el nombre de files del grid.

    Exemple de 3 files:

    HTML
    1
2
3
4
5
    <div class="grid grid-rows-3 gap-4">
  <div class="bg-blue-500 p-4">Fila 1</div>
  <div class="bg-green-500 p-4">Fila 2</div>
  <div class="bg-red-500 p-4">Fila 3</div>
</div>

    En aquest exemple, hem creat un grid amb 3 files i hem afegit un gap-4 per separar les files entre elles.

    4. Utilitzar fr (fraccions) per a les columnes i files

    En lloc de definir columnes i files amb un nombre fix, també pots utilitzar fraccions per repartir l'espai disponible. Tailwind permet utilitzar la unitat fr (fractions) per especificar el fraccionament de les columnes i files.

    Exemple amb fraccions:

    HTML
    1
2
3
4
5
    <div class="grid grid-cols-3 gap-4">
  <div class="bg-blue-500 p-4 col-span-2">Aquesta columna ocupa 2 fraccions</div>
  <div class="bg-green-500 p-4">Columna 2</div>
  <div class="bg-red-500 p-4">Columna 3</div>
</div>
    • col-span-2: Fa que aquesta columna ocupi 2 de les 3 fraccions disponibles de la graella.

    5. Classes de Columnes Responsives

    Tailwind CSS permet definir el nombre de columnes de forma responsiva, és a dir, que el nombre de columnes canvia en funció de la mida de la pantalla.

    Per fer-ho, pots utilitzar les classes responsives com sm:, md:, lg:, xl: per definir el nombre de columnes a diferents mides de pantalla.

    Exemple de columnes responsives:

    HTML
    1
2
3
4
5
6
    <div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
  <div class="bg-blue-500 p-4">Columna 1</div>
  <div class="bg-green-500 p-4">Columna 2</div>
  <div class="bg-red-500 p-4">Columna 3</div>
  <div class="bg-yellow-500 p-4">Columna 4</div>
</div>

    Explicació de les classes responsives:

    • grid-cols-1: Per a pantalles petites (per defecte), hi ha una sola columna.
    • sm:grid-cols-2: A partir de pantalles petites (sm), es faran 2 columnes.
    • md:grid-cols-3: A partir de pantalles mitjanes (md), es faran 3 columnes.
    • lg:grid-cols-4: A partir de pantalles grans (lg), es faran 4 columnes.

    6. Ús de auto per a columnes flexibles

    Si vols que les columnes siguin automàtiques i s’adaptin al contingut, pots utilitzar la classe grid-cols-auto.

    HTML
    1
2
3
4
5
    <div class="grid grid-cols-auto gap-4">
  <div class="bg-blue-500 p-4">Columna 1</div>
  <div class="bg-green-500 p-4">Columna 2</div>
  <div class="bg-red-500 p-4">Columna 3</div>
</div>

    7. Ajustar la mida de les files (grid-rows-auto)

    Pots utilitzar grid-rows-auto per fer que les files s’ajustin automàticament a l’altura del contingut.

    HTML
    1
2
3
4
5
    <div class="grid grid-rows-auto gap-4">
  <div class="bg-blue-500 p-4">Fila 1</div>
  <div class="bg-green-500 p-4">Fila 2</div>
  <div class="bg-red-500 p-4">Fila 3</div>
</div>

    Resum:

    • grid-cols-{n}: Defineix el nombre de columnes (on {n} és el nombre de columnes).
    • grid-rows-{n}: Defineix el nombre de files.
    • grid-cols-auto: Fa que les columnes s’ajustin automàticament al contingut.
    • gap-{n}: Afegeix espai entre les files i columnes.
    • Classes responsives com sm:, md:, lg:, xl: per definir dissenys adaptables a diferents mides de pantalla.

    Amb aquestes classes, tens un control total sobre les columnes i files a Tailwind CSS per crear dissenys flexibles i responsius.
