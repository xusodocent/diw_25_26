    Animacions en CSS

    Keyframes

    Per crear animacions en CSS, utilitzem les regles de @keyframes i les propietats d'animació per definir com ha de canviar l'estil d'un element al llarg del temps.

    Conceptes Bàsics de les Animacions en CSS

    @keyframes:

    • Defineix les etapes d'una animació amb percentatges (0% a 100% o from i to).

    • A cada etapa pots canviar propietats CSS com transform, opacity, background-color, etc.

    Propietat animation:

    • Combina diverses configuracions de l'animació en una sola línia:

      CSS
      1
      animation: <nom_de_keyframe> <durada> <temps_d'espera> <tipus_de_repetició>;

      Exemple:

      CSS
      1
      animation: bounce 2s infinite ease-in-out;

    Propietat transform:

    • Permet transformar l'element visualment (canviar posició, mida, rotació, escala, etc.) amb funcions com:

    translateX, translateY: Moure l'element. scale: Escalar l'element. rotate: Rotar l'element. skew: Inclinar l'element.

    Propietat opacity:

    • Controla la transparència de l'element (0 és invisible, 1 és completament visible).

    Explicació dels Efectes

    Rebot (Bounce)

    Com funciona?

    Utilitza transform: translateY per moure verticalment l'element amunt i avall.

    Codi clau:

    CSS
    1
2
3
4
    @keyframes bounce {
0%, 100% { transform: translateY(0); }
50% { transform: translateY(-50px); }
}

    Rotació (Rotate)

    Com funciona?

    Utilitza transform: rotate per girar l'element en l'eix Z.

    Codi clau:

    CSS
    1
2
3
4
    @keyframes rotate {
0% { transform: rotate(0deg); }
100% { transform: rotate(360deg); }
}

    Pulsació (Pulse)

    Com funciona?

    Utilitza transform: scale per fer créixer i reduir l'element.

    Codi clau:

    CSS
    1
2
3
4
    @keyframes pulse {
0%, 100% { transform: scale(1); }
50% { transform: scale(1.2); }
}

    Desplaçament (Slide)

    Com funciona?

    Utilitza transform: translateX per moure l'element horitzontalment d'esquerra a dreta. Codi clau:

    CSS
    1
2
3
4
5
    @keyframes slide {
0% { transform: translateX(-150%); }
50% { transform: translateX(0); }
100% { transform: translateX(150%); }
}

    Vibració (Shake)

    Com funciona?

    Utilitza transform: translateX amb valors petits i ràpids per moure l'element d'esquerra a dreta com si tremolés.

    Codi clau:

    CSS
    1
2
3
4
5
    @keyframes shake {
0%, 100% { transform: translateX(0); }
25% { transform: translateX(-5px); }
75% { transform: translateX(5px); }
}

    Gir 3D (Flip)

    Com funciona?

    Utilitza transform: rotateY per fer un gir complet al voltant de l'eix Y.

    Codi clau:

    CSS
    1
2
3
4
5
    @keyframes flip {
0% { transform: rotateY(0); }
50% { transform: rotateY(180deg); }
100% { transform: rotateY(360deg); }
}

    Fade

    Com funciona?

    Modifica la propietat opacity per fer que l'element aparegui i desaparegui.

    Codi clau:

    CSS
    1
2
3
4
    @keyframes fade {
0%, 100% { opacity: 1; }
50% { opacity: 0.2; }
}

    Canvi de color (Color-change)

    Com funciona?

    Modifica la propietat background-color per canviar el color de l'element.

    Codi clau:

    CSS
    1
2
3
4
5
    @keyframes color-change {
0% { background: #f093fb; }
50% { background: #f5576c; }
100% { background: #f093fb; }
}

    Com Fer-les Personalitzades

    Pots combinar diverses propietats com transform, opacity, color, border-radius, etc., per crear animacions més elaborades.
