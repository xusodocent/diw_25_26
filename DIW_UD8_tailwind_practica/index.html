    Pràctica Tailwind

    Pràctica de Tailwind CSS: Creació d'una pàgina web responsive

    Objectiu

    Crear una pàgina web responsive utilitzant Tailwind CSS, implementant un disseny modern amb un layout flexible, utilitzant utilitats per a espais, tipografia, colors i estructures responsives.

    Preparació de l'entorn

    Abans d'iniciar, els alumnes han de configurar el seu entorn de treball:

    • Instal·lar Tailwind CSS: Si no tenen Tailwind CSS instal·lat, poden fer-ho a través de CDN per facilitar la pràctica sense necessitat de configuracions avançades. Per començar, només cal afegir el següent al seu arxiu HTML:
    HTML
    1
2
3
    <head>
  <script src="https://cdn.tailwindcss.com"></script>
</head>

    Expliquem els conceptes bàsics

    Aquesta secció és per introduir als alumnes a les utilitats de Tailwind.

    Principals utilitats de Tailwind CSS:

    • Colors: Tailwind té una paleta de colors predeterminada. Exemple: bg-blue-500, text-white, border-gray-300.
    • Tipografia: Exemples: font-semibold, text-xl, text-gray-700.
    • Espaiat: Exemple: m-4 per a marges, p-4 per a padding, space-x-4 per espai horitzontal entre elements.
    • Flexbox i Grid: Exemple: flex, grid, justify-center, items-center.
    • Responsivitat: Utilitzar classes de tailwind responsive com: sm:, md:, lg:, xl: per modificar el disseny a diferents tamanys de pantalla.

    Activitat pràctica

    Enunciat

    Crear una pàgina web responsive que simule una landing page amb un header, secció de contingut, i peu de pàgina (footer). La temàtica de la web serà lliure, podeu fer-la de parcs naturals, cotxes, gastronomia, etc... el que més vos agrade.

    Tasca 1: Crear el layout bàsic

    Creeu un fitxer HTML amb un disseny bàsic. La pàgina ha de tenir un header fixat a la part superior, una secció de contingut amb imatges i text, i un footer.

    Instruccions:

    • Crear un header amb el nom de l'aplicació i un menú de navegació.
    • Crear una secció de contingut amb un títol, text i imatge.
    • Crear un footer amb informació de contacte.

    Tasca 2: Fer-ho responsive

    Ara que tenim el disseny bàsic, feu que la pàgina sigui responsive utilitzant les classes de Tailwind.

    • Fer que el menu de navegació sigui vertical en pantalles petites.
    • Aprofitar el sistema de col·lapse de les seccions utilitzant flexbox o grid per organitzar el contingut de manera flexible.
    • Implementar màquines de media queries per adaptar-se a diferents resolucions (utilitzant classes com sm:, md:, lg:).

    Tasca 3: Personalització

    Els alumnes poden personalitzar el disseny afegint els següents elements:

    • Afegir una banda de color a l'header o al footer.
    • Canviar els tipus de lletra utilitzant les utilitats de Tailwind per a tipografia.
    • Personalitzar els col·locaments i mides de les imatges per a fer-les responsive.

    Activitats Addicionals

    1. Crear un formulari: Demanar als alumnes que creïn un formulari de contacte utilitzant les classes de Tailwind per a inputs, botons, i espaiat.

    2. Afegir un efecte hover a les imatges: Fer que les imatges tinguin un efecte hover (canvi de mida, opacitat, etc.) utilitzant les classes de hover de Tailwind.

    El codi no és la sol·lució

    Les porcions de codi proporcionades només són una ajuda, en cap cas és la sol.lució.
